# 物理公式對照表

以下是常見物理公式的對照整理，方便查詢和快速複習。

## 一、運動學

| 範疇 | 公式 | 說明 |
|---|---|---|
| 速度 | $v = \frac{s}{t}$ | 位移除以時間 |
| 加速度 | $a = \frac{\Delta v}{\Delta t}$ | 速度變化量除以時間 |
| 勻加速直線運動位移 | $s = v_0 t + \frac{1}{2} a t^2$ | 初速、加速度和時間求位移 |
| 末速 | $v = v_0 + a t$ | 初速加上加速度乘時間 |
| 末速平方 | $v^2 = v_0^2 + 2 a s$ | 無時間參數的運動關係 |

## 二、力與牛頓定律

| 範疇 | 公式 | 說明 |
|---|---|---|
| 牛頓第二定律 | $F = m a$ | 力等於質量乘加速度 |
| 重力 | $F_g = m g$ | 地球表面重力加速度 $g \approx 9.8\ \mathrm{m/s^2}$ |
| 彈力（虎克定律） | $F = -k x$ | 彈簧力與伸縮量成正比 |

## 三、功與能量

| 範疇 | 公式 | 說明 |
|---|---|---|
| 功 | $W = F s \cos\theta$ | 力與位移的叉積 |
| 動能 | $K = \frac{1}{2} m v^2$ | 物體因運動具有的能量 |
| 位能 | $U = m g h$ | 重力位能 |
| 彈性位能 | $U = \frac{1}{2} k x^2$ | 彈簧儲存的能量 |
| 能量守恆 | $E_{\text{總}} = K + U$ | 總能量保持不變 |

## 四、動量與衝量

| 範疇 | 公式 | 說明 |
|---|---|---|
| 動量 | $p = m v$ | 質量乘以速度 |
| 衝量 | $J = F \Delta t$ | 力乘以作用時間 |
| 動量守恆 | $m_1 v_{1i} + m_2 v_{2i} = m_1 v_{1f} + m_2 v_{2f}$ | 無外力時總動量不變 |

## 五、圓周運動與萬有引力

| 範疇 | 公式 | 說明 |
|---|---|---|
| 向心加速度 | $a_c = \frac{v^2}{r} = \omega^2 r$ | 速度或角速度表示 |
| 向心力 | $F_c = m \frac{v^2}{r}$ | 保持圓周運動的力 |
| 萬有引力 | $F = G \frac{m_1 m_2}{r^2}$ | 兩質量之間的引力 |

## 六、波與光學

| 範疇 | 公式 | 說明 |
|---|---|---|
| 波速 | $v = f \lambda$ | 波速等於頻率乘波長 |
| 透鏡公式 | $\frac{1}{f} = \frac{1}{d_o} + \frac{1}{d_i}$ | 焦距與物距、像距關係 |
| 折射定律 | $n_1 \sin\theta_1 = n_2 \sin\theta_2$ | 斯涅耳定律 |

## 七、電磁學

| 範疇 | 公式 | 說明 |
|---|---|---|
| 歐姆定律 | $V = I R$ | 電壓等於電流乘電阻 |
| 電功率 | $P = V I = I^2 R = \frac{V^2}{R}$ | 電路功率 |
| 電容 | $C = \frac{Q}{V}$ | 電荷與電壓之比 |
| 電場強度 | $E = \frac{F}{q}$ | 單位電荷所受的力 |
| 電勢差 | $V = \frac{W}{q}$ | 單位電荷的位能差 |

## 八、熱力學

| 範疇 | 公式 | 說明 |
|---|---|---|
| 熱力學第一定律 | $\Delta U = Q - W$ | 內能變化等於熱量減去功 |
| 理想氣體狀態方程式 | $p V = n R T$ | 氣體狀態關係 |
| 熱膨脹 | $\Delta L = \alpha L_0 \Delta T$ | 線膨脹 |

## 九、常用常數

- 重力加速度：$g = 9.8\ \mathrm{m/s^2}$
- 靜電力常數：$k_e = 8.99 \times 10^9\ \mathrm{N\cdot m^2/C^2}$
- 普朗克常數：$h = 6.626 \times 10^{-34}\ \mathrm{J\cdot s}$
- 玻爾茲曼常數：$k_B = 1.38 \times 10^{-23}\ \mathrm{J/K}$
